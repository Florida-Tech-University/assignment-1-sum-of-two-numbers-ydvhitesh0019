# Java Assignment: Sum of Two Numbers

## Objective
Write a Java program that:
- Reads two integers from standard input
- Outputs their sum

## Input
Two integers separated by space or newline.

## Output
A single integer: the sum of the two numbers.

⚠️ Do NOT print extra text such as prompts or labels.

## Example
Input:
3 5

Output:
8
